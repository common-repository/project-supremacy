== Description ==
Cutting edge technology combined with our SEO expertise has brought one of a kind SEO plugin for your WordPress that will not only optimize your website but will boost up your rankings with ease

- Easy WP Setup will enable you to set up your newly created WordPress with blazing speed.
- Apply Global SEO settings all in one place
- Create and manage SEO projects

An amazing collection of powerful SEO tools inside of one WordPress plugin.

== Installation ==
- Go to your WordPress website
- At plugins, Add a New plugin and select a zip file of Project Supremacy V3 Lite (or FTP the extracted zip file into /wp-content/plugins/)
- Activate the plugin
